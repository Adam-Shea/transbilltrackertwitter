# Trans Bill Tracker Twitter

https://twitter.com/TrackTransBills

## Setup

Install node modules with `npm install`
Add in your `.env`
Run `npm run dev` to start the program

# Example .env

```
APP_KEY = ""
APP_SECRET = ""
ACCESS_TOKEN = ""
ACCESS_SECRET = ""
API_KEY = ""
GOOGLE_PRIVATE_KEY = ""
GOOGLE_CLIENT_EMAIL = ""
GOOGLE_SHEET_ID = ""
```
